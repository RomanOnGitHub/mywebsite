# Аудит доступности (A11y)

Проводим комплексный аудит доступности и помогаем сделать ваш сайт доступным для всех пользователей.

## Наши услуги

### Аудит доступности

- Автоматизированное тестирование
- Ручное тестирование
- Тестирование с screen readers
- Проверка соответствия WCAG

### Улучшения

- Исправление проблем доступности
- Оптимизация для assistive technologies
- Улучшение навигации
- Альтернативные форматы контента

### Консультации

- Обучение команды
- Рекомендации по дизайну
- Best practices
- Документация

## Стандарты

- WCAG 2.1 (A, AA, AAA)
- Section 508
- EN 301 549
- ADA compliance
