# Разработка мобильного приложения

## Задача

Клиент нуждался в мобильном приложении для управления заказами с поддержкой iOS и Android.

## Решение

Разработали кроссплатформенное решение:

### Технологический стек

- React Native для кроссплатформенной разработки
- Redux для управления состоянием
- Firebase для бэкенда
- Stripe для платежей

### Ключевые функции

- Управление заказами в реальном времени
- Push-уведомления
- Офлайн-режим
- Биометрическая аутентификация

### UX/UI дизайн

- Современный Material Design
- Интуитивная навигация
- Адаптивная вёрстка
- Темная тема

## Результаты

- **4.8/5** рейтинг в App Store
- **4.7/5** рейтинг в Google Play
- **+200%** активных пользователей
- **+150%** заказов через приложение
