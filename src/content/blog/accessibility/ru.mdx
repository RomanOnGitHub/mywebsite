# Веб-доступность (A11y) - полное руководство

Веб-доступность — это практика создания сайтов, которые могут использовать все люди, независимо от их способностей. Это не только этически правильно, но и юридически требуется во многих странах.

## Что такое A11y?

A11y (Accessibility) включает:

- Визуальные ограничения
- Слуховые ограничения
- Двигательные ограничения
- Когнитивные ограничения

## Стандарты доступности

### WCAG 2.1

Руководящие принципы включают:

- **Воспринимаемость**: Контент должен быть воспринимаемым
- **Управляемость**: Интерфейс должен быть управляемым
- **Понятность**: Информация должна быть понятной
- **Надёжность**: Контент должен быть надёжным

### Уровни соответствия

- **A**: Минимальный уровень
- **AA**: Рекомендуемый уровень
- **AAA**: Высший уровень

## Практические рекомендации

### Семантический HTML

Используйте правильные теги:

- `<header>`, `<nav>`, `<main>`, `<article>`
- Правильная иерархия заголовков
- ARIA-атрибуты где необходимо

### Клавиатурная навигация

Обеспечьте:

- Tab-навигацию
- Видимый focus
- Логический порядок элементов
- Skip links

### Альтернативный текст

Все изображения должны иметь:

- Описательный alt-текст
- Пустой alt для декоративных изображений
- Текстовые альтернативы для графиков
