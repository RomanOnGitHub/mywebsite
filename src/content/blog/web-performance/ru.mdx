# Оптимизация производительности веб-сайтов

Производительность сайта напрямую влияет на пользовательский опыт, конверсии и SEO-рейтинг. В этой статье мы рассмотрим ключевые техники оптимизации.

## Метрики производительности

### Core Web Vitals

Google использует следующие метрики:

- **LCP (Largest Contentful Paint)**: Время загрузки основного контента
- **FID (First Input Delay)**: Задержка первого взаимодействия
- **CLS (Cumulative Layout Shift)**: Стабильность визуального контента

### Целевые значения

Стремитесь к следующим показателям:

- LCP < 2.5 секунды
- FID < 100 миллисекунд
- CLS < 0.1

## Техники оптимизации

### Оптимизация изображений

- Используйте современные форматы (WebP, AVIF)
- Ленивая загрузка (lazy loading)
- Responsive images
- CDN для статики

### Минификация кода

- Минификация CSS и JavaScript
- Tree shaking
- Code splitting
- Bundle optimization

### Кэширование

- Browser caching
- CDN caching
- Service Workers
- HTTP/2 и HTTP/3

## Инструменты

Популярные инструменты для анализа:

- Lighthouse
- WebPageTest
- Chrome DevTools
- GTmetrix
