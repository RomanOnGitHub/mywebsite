# GitHub поисковые запросы для графов знаний

## Основные запросы

### 1. Knowledge Graph / Граф знаний
```

knowledge graph astro
knowledge graph d3
knowledge graph force-directed
```

### 2. Force-directed graphs
```
force-directed graph astro

```

### 3. Content graph / Content relationships

markdown link graph

```

### 4. Astro-specific

```

### 6. Semantic networks

## Рекомендуемые репозитории для изучения

1. **Obsidian Graph View** - классический пример графа знаний
   - `obsidianmd/obsidian-core`
   - Поиск: `obsidian graph view`

2. **Roam Research** - граф связей между заметками
   - Поиск: `roam research graph`

3. **Foam** - VS Code extension с графом знаний
   - Поиск: `foam graph visualization`

4. **Astro examples** - официальные примеры Astro
   - `withastro/astro/examples`
   - Поиск: `astro examples graph`

5. **D3.js examples** - примеры визуализаций
   - `d3/d3-force examples`
   - Поиск: `d3 force graph example`

## Фильтры для GitHub поиска

### По языку
- `language:typescript`
- `language:javascript`
- `language:astro`

### По звёздам (популярные)
- `stars:>100`
- `stars:>500`

### По дате обновления
- `pushed:>2024-01-01`

### Комбинированный пример
```
language:typescript stars:>50 astro graph visualization
```

## Что искать в коде

После скачивания репозитория, обрати внимание на:

1. **Структуру данных графа**
   - Как хранятся nodes и edges
   - Формат JSON/GraphQL
   - Связи между контентом

2. **Визуализацию**
   - Используемая библиотека (force-graph, d3-force, vis-network)
   - Стилизация рёбер и узлов
   - Интерактивность (hover, click, drag)

3. **Фильтрацию и поиск**
   - Как реализованы фильтры
   - Поиск по графу
   - Выделение путей

4. **Производительность**
   - Оптимизация для больших графов
   - Ленивая загрузка
   - Виртуализация

5. **Интеграция с Astro**
   - Как генерируются данные графа
   - Build-time vs runtime
   - SSR/SSG considerations
